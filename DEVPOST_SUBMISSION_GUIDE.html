Project Name

Nirvan - AI Life Coach

Tagline / Short Description

Minimalist AI coaching in your pocket. 6 specialized coaches, zero setup, guidance at the right moment. Built for Simon's audience.

Simon's audience loves productivity, great design, and building systemsâ€”and he wants to bring coaching to more people. I wanted to create a clean, minimal mobile app where users can get instant, expert-level coaching across fitness, career, mindfulness, relationships, finance, and productivityâ€”all in one place, with zero complexity.

What it does

Nirvan is a minimalist AI coaching platform that provides 6 specialized AI coaches, each trained for specific life domains. Unlike generic AI assistants, each coach has a distinct personality, expertise, and coaching methodology.

The 6 Coaches:

  • Fitness Coach: Personalized workout plans, nutrition guidance, and progress tracking
  • Career Coach: Resume optimization, interview prep, salary negotiation, and career transitions
  • Mindfulness Coach: Stress management, meditation techniques, and mental wellness
  • Relationship Coach: Communication skills, conflict resolution, and healthy relationships
  • Finance Coach: Budgeting, investing, debt management, and financial planning
  • Productivity Coach: Time management, goal setting, and habit formation

Key Features:

  • Zero Setup: No account creation, no complex onboardingâ€”just open and start chatting
  • Clean Design: Minimalist interface with thoughtful spacing and smooth animations
  • Seamless Switching: Move between coaches throughout your day with a single tap
  • 24/7 Availability: Get guidance at the right moment, whenever you need it
  • Privacy-First: Conversations stored locally on your device, not in the cloud
  • Freemium Model: 5 free messages per coach, then $9.99/month for unlimited access

How we built it

Frontend: Built with React Native (v0.74.5) and Expo SDK (v51) for cross-platform mobile development. Used Expo Router for file-based navigation and TypeScript for type safety. The minimalist UI uses React Native StyleSheet with custom design tokens, React Native Reanimated for smooth 60fps animations, and thoughtful spacing that creates a premium, uncluttered experience.

AI Integration: Powered by OpenAI's GPT-4 API. Each coach has a specialized system prompt that defines its personality, expertise, and coaching methodology. The AI service handles conversation context, maintains chat history, and delivers expert-level responses tailored to each domain.

Monetization: Integrated RevenueCat SDK for seamless subscription management. Configured with a freemium model (5 messages per coach free, then $9.99/month for unlimited access). RevenueCat handles receipt validation, entitlement management, and cross-platform subscription syncing.

Data Storage: Used AsyncStorage for local data persistence. All conversations are stored on-device for complete privacyâ€”no cloud storage, no analytics, no data sharing.

CI/CD: Automated iOS builds using GitHub Actions and EAS Build. The workflow handles dependency installation, environment variable injection, app building, and TestFlight deploymentâ€”all triggered via workflow dispatch.

Challenges we ran into

RevenueCat Configuration: Initially encountered Error 23 due to missing App Store Connect configuration. Had to properly set up the App-Specific Shared Secret, configure products in both App Store Connect and RevenueCat, and create the correct offering structure. The debugging process taught me the importance of matching product IDs and bundle identifiers exactly.

AI Prompt Engineering: Creating distinct personalities for each coach required extensive prompt engineering. Each coach needed to feel like a real expert in their domain while maintaining a consistent, helpful tone. Balancing specificity with flexibility was challengingâ€”coaches needed to provide actionable advice without being overly prescriptive.

Minimalist Design Implementation: Achieving a truly minimalist design that still felt feature-rich required careful attention to spacing, typography, and animation timing. Every pixel matteredâ€”too much whitespace felt empty, too little felt cluttered. Finding the right balance took multiple iterations.

State Management: Managing conversation state across multiple coaches, tracking message counts for the freemium model, and syncing premium status required careful context architecture. Used React Context API to avoid prop drilling while keeping the state logic clean and maintainable.
